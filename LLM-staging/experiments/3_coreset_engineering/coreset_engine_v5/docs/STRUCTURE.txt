ARCHITECTURE LAYERS
═══════════════════════════════════════════════════════════════════════════════

Layer 1: Configuration Management
  → config.py: Type-safe configuration
  → Types: DeduplicationConfig, DiversityConfig, SelectionConfig, etc.

Layer 2: Data Loading & Validation
  → loaders.py: ChunkLoader
  → types.py: ChunkMetadata
  → curriculum/loader.py: CurriculumLoader

Layer 3: Preprocessing (Deduplication + Scoring)
  → deduplicator.py: ExactDeduplicator, NearDeduplicator
  → scorer.py: DiversityScorer, TokenFrequencyAnalyzer

Layer 4: Selection Algorithm
  → engine.py: SelectionEngine
  - Stratified bucketing
  - Density-aware sampling
  - Protected slice enforcement

Layer 5: Output & Reporting
  → loaders.py: CoresetWriter, AblationReporter
  → types.py: CoresetManifest
