📖 QUICK START

Basic usage (with checkpointing - recommended):

  python coreset_builder_optimized.py \
      --config config/pipeline.yaml \
      --curriculum config/curriculum.yaml \
      --checkpoint-dir ./checkpoints

If interrupted, simply restart the same command:
  - Automatically finds last checkpoint
  - Skips already-processed batches
  - Resumes from next batch
  - Zero data loss
